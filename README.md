# TypeScript Express Sample
The main purpose of this repository is to start a node server in TypeScript.
这个仓库主要用来学习如何使用Typescript+Express开发node服务,用于个人学习。
项目开始，采用TypeScript Node Starter构建项目，并去除其中的view层。
